# delete_video()

> Remove a video by ID

Remove a video by ID.

```python theme={null}
client.delete_video("video_id")
```

**Parameters:**

| Parameter  | Type  | Description                          |
| ---------- | ----- | ------------------------------------ |
| `video_id` | `str` | ID of the video to delete (required) |

**Returns:** Dict with deletion status
